Engineering Details

Strict RAG Architecture against Hallucinations

While connecting the RAG pipeline to MongoDB Atlas was straightforward, the real challenge was deploying the model locally and ensuring zero hallucinations for academic use cases.

I utilized MongoDB Atlas's Vector Search (using Cosine Similarity) to retrieve the top relevant document chunks. The core effort went into designing highly restrictive prompting for Llama 3.1 to only generate responses based on retrieved chunks.
